Quem é Jaime?

Jaime é um launcher criado para pessoas amantes de uma area de trabalho limpa, pessoas que não gostam de perder tempo procurando um ícone com o mouse só para abrir um programa, arquivo ou até mesmo uma pasta. Veja algumas das coisas que o Jaime pode fazer por você:


Adicione pastas e arquivos para um acesso rápido.

Abra rapidamente qualquer sistema instalado em sua máquina.

Abra urls em seu browser diretamente pelo Jaime.

Cadastre motores de busca para facilitar suas pesquisas na internete.


Substitua o atalho Win+R e execute comandos diretamente no Jaime.

Resolva contas e expressões matemáticas simples sem precisar abrir a calculadora.

Traga a barra de favoritos do Chrome e Firefox para o Jaime.

Crie notas rapidamente em arquivos de texto.

Como funciona?

O Jaime é bem simples de utilizar, basta abrir, configurar e digitar! Porém o Jaime é seu amigo e explicará exatamente como tudo funciona nos tutoriais abaixo.


Como instalar?

Caso sua máquina não possua a última versão do .Net Framework instalado basta baixar e instalar através DESTE LINK e instalar seguindo as instruções do instalador. Após instalar o .Net Framework, basta extrair o arquivo baixado aqui mesmo no site e executar o arquivo jaime.exe, seguir as instruções do instalador e depois executar o ícone que foi criado na área de trabalho , este aqui com o ícone do bigodinho.

Configurando o Jaime

Jaime não possui configurações complicadas pois foi feito com um objetivo definido de ser simples porém muito eficiênte.

Configurações de motores de busca

Configure aqui os motores de busca que deseja que o Jaime pesquise.

Clique na imagem para ampliar

  1. Url do motor de busca (O parametro padrão para a busca é {term}, o restante deve ser adicionado com numeração em ordem. Ex: url.com.br/{0}/{1}/{term}).
  2. Parâmetros da url (Os parâmetros devem ser adicionados separados por ponto e vírgula. Ex: parametro 0; parametro 1; parametro 2).
  3. Comando de ativação do motor de busca (Configure GBR para buscas no google do Brasil e sempre que precisar pesquisar digite: GBR+ESPAÇO+O QUE DESEJA PESQUISAR Ex: 'gbr Por que o Jaime é tão bom comigo? =~)').
  4. Botões para adicionar ou excluir motores de busca.
  5. Grid de visualização e seleção dos motores de busca.
